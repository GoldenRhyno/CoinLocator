
## Running coin-promoting Locally Using LTS node version

Clone Repo:

```sh
git clone https://github.com/GoldenRhyno/CoinLocator
cd CoinLocator
```

Install Dependencies:

```sh
npm install
```

Run The Application:

```sh
npm start
```
